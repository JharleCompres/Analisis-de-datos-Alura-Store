# Analisis-de-datos-Alura-Store
Empleamos un análisis de datos acerca de las ventas de 4 tiendas de un cliente el cual nos hizo la peticion de que quiere saber que tienda vender. Donde conoceremos que tienda tiene el menor rendiemiento en cuánto a ventas y nosotros como cientifico de datos proporcionar nuestra recomendacion al cliente de que tienda debe vender
